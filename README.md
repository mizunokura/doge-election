# ヴェネツィア総督選挙

1268年のドージェ（総督）選挙プロトコルに基づく戦略シミュレーションゲーム。

**[ブラウザで遊ぶ](https://mizunokura.github.io/doge-election/)**

## 概要

あなたはヴェネツィアの野心的な貴族。買収と政治工作で自分の候補をドージェに当選させよ。ただし金貨の無駄遣いは命取り——くじ引きラウンドで買収した委員が落ちることもある。

## ルール

全10ラウンド。くじ引きと選挙を交互に繰り返し、最終的に41名の委員会から25票の承認を得た候補がドージェに選出される。

| ラウンド | 種別 | 委員会 | 承認数 |
|---------|------|--------|-------|
| 1 | くじ引き | 全員 → 30名 | - |
| 2 | くじ引き | 30 → 9名 | - |
| 3 | 選挙 | 9名が40名を選出 | 7 |
| 4 | くじ引き | 40 → 12名 | - |
| 5 | 選挙 | 12名が25名を選出 | 9 |
| 6 | くじ引き | 25 → 9名 | - |
| 7 | 選挙 | 9名が45名を選出 | 7 |
| 8 | くじ引き | 45 → 11名 | - |
| 9 | 選挙 | 11名が41名を選出 | 9 |
| 10 | 選挙 | 41名がドージェを選出 | 25 |

- **くじ引きラウンド**: ランダムに委員が選ばれる。買収の余地なし
- **選挙ラウンド**: 投票前に金貨を使って中立派や敵派閥を買収できる
- **スコア**: 勝利ボーナス(1000) + 残り金貨 × 10
